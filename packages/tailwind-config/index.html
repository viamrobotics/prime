@viamrobotics/tailwind-config

@viamrobotics/tailwind-config is Viam’s shared Tailwind CSS v4 configuration. It exposes design tokens as Tailwind v4 theme values in an @theme {} block plus an optional ./fonts entry point that side-effect imports the variable-font packages.

Requires Tailwind v4

A Tailwind v4 bundler integration is required (@tailwindcss/vite or @tailwindcss/postcss). v3 is not supported.

What it ships

Section titled “What it ships”
  • tailwind-config.css — the @theme {} block: colors, spacing, fonts, shadows, z-index tokens.
  • fonts.js — three import '@fontsource-variable/*' lines for Space Grotesk, Roboto Mono, and Public Sans.

Token values match the Viam design system. Don’t invent values; copy from the published guides.
