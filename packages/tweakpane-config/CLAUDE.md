# @viamrobotics/tweakpane-config

Viam-themed configuration for [`svelte-tweakpane-ui`](https://kitschpatrol.com/svelte-tweakpane-ui). Exposes a preset `Theme` matching the Viam design system; consumers apply it via `svelte-tweakpane-ui`'s own `ThemeUtils` API.

## Surface

- `primeTheme` — the Prime `Theme` object. Pass it to `ThemeUtils.setGlobalDefaultTheme(...)` or to any component's `theme` prop. See the [`svelte-tweakpane-ui` themes docs](https://kitschpatrol.com/svelte-tweakpane-ui/docs/themes) for the usage patterns.
- `Theme` — re-exported from `svelte-tweakpane-ui` for convenience.

## Layout

- `src/lib/` — published source. `src/lib/index.ts` re-exports the public API.
- `src/lib/theme.ts` — the Prime theme object (plain hex/rgba values).
- `src/routes/` — SvelteKit playground app. Built statically and embedded under [apps/docs](../../apps/docs/) at `/playground/tweakpane-config/`. Not published to npm.
- `dist/` — generated by `tsc -p tsconfig.build.json` during `pnpm build`; what npm sees via the `files` field.

## Peer dependencies

- `svelte ^5`
- `svelte-tweakpane-ui ^1.5.16`

## Conventions

- Theme hex values mirror tokens in [`@viamrobotics/tailwind-config`](../tailwind-config/CLAUDE.md). The values are inlined rather than referenced via `var(--color-*)` because `svelte-tweakpane-ui` doesn't reliably resolve CSS var references inside its theme values. If you change a token in `tailwind-config.css`, update [src/lib/theme.ts](./src/lib/theme.ts) to match.
- Token values must match the Viam design system at https://design.viam.com/. Don't invent values; copy from the published guides.
- This package is data-only: it exports a theme object and a type. The decision of _where_ and _when_ to call `ThemeUtils.setGlobalDefaultTheme` belongs to the consumer (SSR concerns, lazy initialization, etc.). Resist the urge to add a helper that wraps it.
- The playground's `+layout.svelte` calls `ThemeUtils.setGlobalDefaultTheme` inside `$effect` so it only runs on the client (tweakpane touches `window`).
- Fonts are not bundled. Consumers load them separately via `@viamrobotics/tailwind-config/fonts` or `@fontsource-variable/public-sans`.

## Commands

```bash
pnpm --filter @viamrobotics/tweakpane-config dev      # start playground dev server
pnpm --filter @viamrobotics/tweakpane-config build    # tsc library + vite build playground
pnpm --filter @viamrobotics/tweakpane-config check    # svelte-check
```
