Viam Prime

Component libraries and design system documentation for Viam apps.
Explore prime-ui Open the prime-ui playground

Prime is Viam’s design-system implementation: a set of npm packages that share design tokens (@viamrobotics/tailwind-config) and ship Svelte 5 components on top of them (@viamrobotics/prime-ui). This site documents what’s available and embeds a live playground for each library package.

Packages

Section titled “Packages”

@viamrobotics/prime-ui

Viam’s Svelte 5 component library. See the package docs for usage and component references, or the live playground to test out components.

@viamrobotics/tailwind-config

Shared Tailwind CSS v4 design tokens and fonts. See the package docs for usage and token references.

@viamrobotics/tweakpane-config

Viam-themed preset for svelte-tweakpane-ui. See the package docs for usage, or the live playground to see examples.
