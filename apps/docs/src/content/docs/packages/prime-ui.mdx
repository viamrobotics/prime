`@viamrobotics/prime-ui` is Viam's Svelte 5 component library. Components are built on top of design tokens from [`@viamrobotics/tailwind-config`](../tailwind-config/) and styled with Tailwind CSS v4.

<Aside type="caution" title="Pre-1.0">
	The v2 of prime is under active development. The public API is unstable; expect breaking changes
	until 1.0.
</Aside>

## Installation

<Steps>

1. Install the package and its peers:

   <Tabs syncKey="pkg">
   	<TabItem label="pnpm">
   		<Code
   			code="pnpm add @viamrobotics/prime-ui @viamrobotics/tailwind-config svelte tailwindcss"
   			lang="sh"
   		/>
   	</TabItem>
   	<TabItem label="npm">
   		<Code
   			code="npm install @viamrobotics/prime-ui @viamrobotics/tailwind-config svelte tailwindcss"
   			lang="sh"
   		/>
   	</TabItem>
   	<TabItem label="yarn">
   		<Code
   			code="yarn add @viamrobotics/prime-ui @viamrobotics/tailwind-config svelte tailwindcss"
   			lang="sh"
   		/>
   	</TabItem>
   </Tabs>

2. In your app's main CSS, import the Tailwind config so token classes resolve:

   <Code code="@import '@viamrobotics/tailwind-config';" lang="css" />

</Steps>
