`@viamrobotics/tailwind-config` is Viam's shared Tailwind CSS v4 configuration. It exposes design tokens as Tailwind v4 theme values in an `@theme {}` block plus an optional `./fonts` entry point that side-effect imports the variable-font packages.

<Aside title="Requires Tailwind v4">
	A Tailwind v4 bundler integration is required (`@tailwindcss/vite` or `@tailwindcss/postcss`). v3
	is not supported.
</Aside>

## Installation

<Steps>

1. Install the package alongside Tailwind:

   <Tabs syncKey="pkg">
   	<TabItem label="pnpm">
   		<Code code="pnpm add --save-dev tailwindcss @viamrobotics/tailwind-config" lang="sh" />
   	</TabItem>
   	<TabItem label="npm">
   		<Code code="npm install --save-dev tailwindcss @viamrobotics/tailwind-config" lang="sh" />
   	</TabItem>
   	<TabItem label="yarn">
   		<Code code="yarn add --dev tailwindcss @viamrobotics/tailwind-config" lang="sh" />
   	</TabItem>
   </Tabs>

2. Import it from your app's main CSS:

   <Code code="@import '@viamrobotics/tailwind-config';" lang="css" />

3. (Optional) Load the variable fonts by adding a side-effect import in your app's entry point:

   <Code code="import '@viamrobotics/tailwind-config/fonts';" lang="ts" />

</Steps>
