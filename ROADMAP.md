# Prime v2 Roadmap

This document tracks the work to bring Prime to v2. The headline change is migrating our component library back into this repo, leaning on headless component primitives wherever possible.

## Stage 1: Component Evaluation

For each component currently in the library, decide whether it should be carried forward into v2 and, if so, what it should be replaced by (a headless primitive, another existing component, or itself).
